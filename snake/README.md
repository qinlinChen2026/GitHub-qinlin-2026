# 贪吃蛇小游戏 (Snake Game)

一个使用Python和Pygame库实现的经典贪吃蛇游戏。

## 功能特点

- 经典的贪吃蛇游戏玩法
- 蛇头有眼睛，会根据移动方向变化
- 食物看起来像苹果
- 网格背景
- 得分系统（当前得分和最高分）
- 游戏结束画面，可重新开始或退出
- 蛇撞墙后游戏结束
- 蛇不能直接反向移动

## 系统要求

- Python 3.6 或更高版本
- Pygame 库

## 安装步骤

### 1. 安装Python
确保你的系统已安装Python 3.6或更高版本。你可以通过以下命令检查：

```bash
python3 --version
```

### 2. 安装Pygame
如果你还没有安装Pygame，可以使用pip安装：

```bash
pip install pygame
```

或者

```bash
python3 -m pip install pygame
```

如果你遇到权限问题，可以尝试：

```bash
pip install --user pygame
```

## 运行游戏

1. 确保你在包含 `snake_game.py` 文件的目录中
2. 运行以下命令：

```bash
python3 snake_game.py
```

或者

```bash
python snake_game.py
```

## 游戏控制

- **方向键上 (↑)** - 向上移动
- **方向键下 (↓)** - 向下移动  
- **方向键左 (←)** - 向左移动
- **方向键右 (→)** - 向右移动
- **R键** - 游戏结束后重新开始
- **Q键** - 游戏结束后退出游戏
- **关闭窗口** - 退出游戏

## 游戏规则

1. 使用方向键控制蛇的移动方向
2. 蛇不能直接反向移动（例如：向右移动时不能立即向左移动）
3. 吃到红色食物（苹果）后，蛇的长度增加，得分增加10分
4. 如果蛇撞到自己，游戏结束
5. 蛇撞墙后游戏结束
6. 游戏结束后，按R键重新开始，按Q键退出

## 游戏界面说明

- **蓝色方块** - 蛇头
- **绿色方块** - 蛇身
- **红色方块** - 食物（苹果）
- **灰色网格** - 游戏区域网格
- **左上角** - 当前得分
- **右上角** - 最高分

## 故障排除

### 常见问题

1. **"ModuleNotFoundError: No module named 'pygame'"**
   - 解决方案：按照上面的安装步骤安装Pygame库

2. **游戏运行缓慢或卡顿**
   - 可以调整 `FPS` 变量值（第13行）来改变游戏速度
   - 增加FPS值使游戏更快，减少FPS值使游戏更慢

3. **游戏窗口无法关闭**
   - 按Q键或点击窗口关闭按钮

4. **方向控制不响应**
   - 确保没有其他程序占用键盘输入
   - 确保游戏窗口是活动窗口

### 自定义游戏设置

你可以在代码中修改以下常量来自定义游戏：

- `WIDTH, HEIGHT` (第9行): 游戏窗口大小
- `GRID_SIZE` (第10行): 网格大小（蛇和食物的大小）
- `FPS` (第13行): 游戏帧率，控制游戏速度
- 颜色常量 (第16-21行): 可以修改游戏颜色

## 代码结构

- `Snake` 类: 处理蛇的移动、生长和绘制
- `Food` 类: 处理食物的生成和绘制
- `draw_grid()`: 绘制游戏网格
- `draw_score()`: 绘制得分显示
- `draw_game_over()`: 绘制游戏结束画面
- `main()`: 主游戏循环和事件处理
